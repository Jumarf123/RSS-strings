# üá™üá≥ English

## Overview

**RSS-strings** is a Windows utility that scans the memory of a selected process and searches for up to **50000** specified strings.

## Features

- Select a running process from the top panel
- Paste a list of strings to search (up to 50000)
- Adjustable search settings
- Results preview on the right side
- Copy / download results for later use

## Requirements

- Windows 10/11
- (Recommended) Run as Administrator for better access to protected processes

## How to use

1. **Run** the program.
2. **Select a process** in the top panel.
3. On the **left**, paste your list of strings.
4. (Optional) Adjust search settings.
5. Click **Scan**.
6. Results will appear on the **right**. You can **copy** or **download** them.

## Notes

- Some processes may not be accessible without elevated permissions.
- Scanning large processes may take longer depending on settings and system load.
