# RSS-strings

Утилита **RSS-strings** *сканирует память* выбранного процесса Windows и ищет до 5000 заданных строк.


**Как пользоваться?**

→ Запустить программу

→ выбрать процесс в верхней панели

→ слева вставить список строк (при необходимости отрегулируй настройки поиска → нажми «Сканировать»;

→ результаты появятся справа, результат вы можете скопировать/скачать или просто посмотреть.
